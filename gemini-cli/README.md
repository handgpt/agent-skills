# Gemini CLI Skills

Reserved location for Gemini CLI specific skills, prompts, wrappers, and install
helpers.

This directory is intentionally separate from `codex/` so Gemini CLI assets can
evolve without inheriting Codex-specific assumptions about skill format, install
location, or workflow hooks.

## Status

Planned. Runtime-specific assets are not published yet.

## Intended Scope

- Gemini CLI workflows
- Gemini CLI install helpers
- Gemini CLI specific prompt or task conventions
